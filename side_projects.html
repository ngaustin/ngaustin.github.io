Side Projects

In my freetime, I love making computers do (borderline) stupid things. Check some of my projects below.

ML-Based Picture Reconstruction Using MS Paint

This project used a combination of computer vision (CV) filtering techniques, basic ML techniques (K-nearest neighbors), and mouse event API's to enable autonomous generation of images through MS Paint (of course given a source image to replicate). This was a project that was more interesting than practical, but I had a fun time showing people images of themselves as replicated by MS paint.

Translating Hand Gestures to Mouse Events

Another Covid-19 project, I decided to use a combination of computer vision (CV) and a general mouse event API to allow hand gestures and movements to guide cursor movements and action inputs. Definitely was not reliable enough to be deployable, but was a fun enough project to realize how difficult it truly is to make computer use accessible to different demographics.
